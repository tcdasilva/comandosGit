Configurar usuario e email do comite >> No Git Bash >>  comando << git user.name "Thiago Silva" >> e comando << git user.email "tcdasilva.ti@gmail.com" >>

Status dos arquivos comando << git status >>

Inicializar o Git >> Na pasta do projeto >> comando << git init >>  

Adicionar modificações >> Na pasta do projeto >> comando << git add nomeDoArquivo ou git add -A (paratadicionar todos) >> 

Commit alterações/arquivos >> comando << git commit -m "Comentarios de adições ou modificações" >>

Logs >> comando << git log >>

Adicionar o endereço de repositorio na nuvem >> comando << git remote add origin https://github.com/tcdasilva/comandosGit.git >>

Verificar o endereço do repositorio nuvem apontado  >> comando << git remote -v >>

Enviar para repositorio nuvem >> comando << git push -u origin master >>

Criar um Ramo >> comando << git branch novosComandos >>

Entrar no novo Ramo >> comando << git checkout novosComandos >>
